Help Page

This application gauges the real-time mood of Twitter users on any topic based on related keywords.

A Twitter account and Internet access is required in order to analyze topics with this app. You may change Twitter accounts from the settings menu by selecting "Change User". Changing accounts will not alter the contents that you have saved within the app.

Disclaimer

The user is responsible for monitoring their own data usage. Analyzing topics with Social Mood Swing may utilize a large amount of data. We recommend that you use Wi-Fi while running analysis sessions.

Keyword Selection Tips

The performance of this application is highly dependent on keywords that you choose. The following tips may be helpful in selecting keywords for various topics:
  • Make sure that your keywords are spelled correctly (keywords are not case sensitive).
  • For Example: The Keyword "Twitter":
    • will match any of the following: TWITTER, twitter, "Twitter", twitter., #twitter, @twitter, or http://twitter.com
    • but it will not match: TwitterTracker or #newtwitter
  • Do not choose keywords that are vague or not specific to the topic, as this may cause the application to gather irrelevant Tweets.
  • Do not choose keywords with built-in bias. Choosing keywords that are negatively/positively associated with the topic to begin with may bias your results.
  • A single keyword field may contain multiple words.

Application Usage

Topics Page
  • When looking at the Topics page, click the "New Topic" button in order to create a new topic.
  • Once there is at least one topic, you may click the pencil icon to edit or delete that topic.
  • Clicking on a topic's title will take you to the "Run History" page for that topic.

 Run History Page
  • From the top of the Run History page you can start a new analysis session by entering the number of hours/minutes you want the session to run for.
  • Starting an analysis session from this page will begin an analysis session based on the keywords for the current topic.
  • General average statistics are displayed concerning all analysis sessions in the history for the current topic.
  • Once at least two sessions are stored in the current topic's history, a graph will be displayed showing the trends for the topic from one analysis session to the next.
  • Tapping on a point in the graph will display the average sentiment and number of Tweets processed for that analysis session.
  • Once there are more than 8 analysis sessions in the current topic's history, the graph will become horizontally scrollable.
  • The date and time of the analysis session is displayed underneath the graph below each point.

 Analysis Session Page
  • At the top of the Analysis Session page is a gauge which displays the ratio of positive vs. negative sentiment for the most recent Tweets that have been processed.
  • Also displayed are the number of Tweets that have been processed as well as the total average sentiment for the current analysis session.
  • Popular Tweets will be displayed below the gauge as they are processed and may be clicked on for more detailed information (this information may be exited by clicking anywhere on the screen).
  • The amount of time remaining for the current analysis session is displayed near the bottom of the screen.
  • Upon clicking "Stop" you will have the option to terminate the current analysis session and choose whether or not to save the current results, or to continue running the session.
  • When an analysis session has ended after the specified time interval, a dialog will display a summary of the session results. You may choose whether or not to save these results.
